# 银行战略决策平台 - UI设计摘要

**生成日期**：2026年3月18日

---

## 设计原则

数据驱动、信息分层、决策效率、专业稳重

## 色彩规范

### 浅色主题
- 主色：深海蓝 `#1E3A5F`
- 辅色：尊享金 `#C9A962`
- 背景：冰灰 `#F5F7FA`
- 卡片：纯白 `#FFFFFF`

### 深色主题
- 主色：午夜蓝 `#1E3A5F`
- 辅色：璀璨金 `#D4B87A`
- 背景：深墨 `#0F172A`

## 核心页面

| 页面 | 核心元素 |
|------|----------|
| 首页/驾驶舱 | KPI四象限、战略目标环形图、时间线 |
| 战略决策看板 | 树形图、时间轴、里程碑 |
| 经营分析仪表盘 | 仪表盘、趋势图、饼图、柱状图、数据表格 |
| 协同工作台 | 待办任务、审批流程、文档协作 |
| 风险监控中心 | 热力地图、风险仪表、预警列表 |
| 市场洞察 | 经济图表、雷达图、竞品卡片 |
| 个人中心 | 日程、消息、设置 |

## 设计亮点

- 深色/浅色主题一键切换
- 4px 网格间距系统
- ECharts 图表配置
- WCAG AA 可访问性
- 响应式断点覆盖移动端到 4K 屏
