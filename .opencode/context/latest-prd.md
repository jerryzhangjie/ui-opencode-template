# 银行战略决策平台 - 需求摘要

**生成日期**：2026年3月18日

---

## 产品概述

面向银行总行高管（行长、副行长、部门总经理）的战略决策与经营协同工作平台，帮助高管层实时掌握全行业务经营数据、审批重大战略决策事项、跨部门协同工作、洞察市场趋势和竞争态势。

## 核心功能模块

| 模块 | 优先级 | 描述 |
|-----|-------|------|
| 战略决策看板 | P0 | 展示银行整体战略目标、执行进度、关键成果 |
| 经营分析仪表盘 | P0 | 资产负债、利润收入、风险管理等核心指标分析 |
| 协同工作台 | P0 | 跨部门任务协同、审批流程、文档共享 |
| 风险监控中心 | P0 | 信用风险、市场风险、操作风险监控预警 |
| 市场洞察 | P1 | 宏观经济、行业动态、竞争对手分析 |
| 会议与日程 | P1 | 高管会议管理、重要日程提醒 |

## 用户权限体系

- 行长：全功能访问，包括所有审批权限
- 副行长：分管条线数据访问，审批分管领域事项
- 部门总经理：本部门数据访问，审批本部门事项

## 技术要求

- 企业AD域集成认证 + 短信/邮箱验证码二次认证
- 基于RBAC的权限控制
- 全链路HTTPS加密 + 审计日志
- 支持深色/浅色主题切换
